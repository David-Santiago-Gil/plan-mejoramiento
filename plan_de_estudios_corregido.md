### **OBJETIVO GENERAL**

Fortalecer las competencias en **programación** JavaScript mediante el desarrollo de ejercicios prácticos que cubran los temas fundamentales evaluados, asegurando la comprensión y aplicación correcta de cada concepto.

### **ACTIVIDADES DE MEJORAMIENTO POR TEMÁTICA**

**I. VARIABLES, TIPOS Y FUNCIONES DE I/O** (10 Ejercicios)

*   **Temas a desarrollar:**
    *   Declaración y uso de variables (`let`, `const`)
    *   Tipos de datos primitivos
    *   Conversión de tipos (`parseInt`, `Number`, `String`)
    *   Funciones de entrada/salida (`prompt`, `alert`, `console.log`)
    *   Coerción de tipos en JavaScript
*   **Actividades específicas:**
    *   Crear 5 programas que utilicen `prompt()` para entrada de datos y `alert()` para mostrar resultados.
    *   Desarrollar 3 ejercicios que demuestren el uso de conversión de tipos de datos.
    *   Implementar 2 programas que ejemplifiquen la diferencia entre `let` y `const`.

**II. ARRAYS Y MANIPULACIÓN DE DATOS** (10 Ejercicios)

*   **Temas a desarrollar:**
    *   Creación y manipulación de arrays
    *   Métodos básicos: `push`, `pop`, `shift`, `unshift`
    *   Métodos de transformación: `map`, `filter`, `reduce`
    *   Iteración con bucles (`for`, `forEach`)
    *   Manipulación de matrices (arrays bidimensionales)
*   **Actividades específicas:**
    *   Desarrollar 4 ejercicios utilizando métodos de modificación de arrays.
    *   Crear 3 programas que implementen métodos de transformación (`.map`, `.filter`).
    *   Implementar 3 ejercicios con bucles para recorrer y manipular arrays.

**III. FUNCIONES Y ESTRUCTURAS CONDICIONALES** (10 Ejercicios)

*   **Temas a desarrollar:**
    *   Declaración de funciones (tradicionales y arrow functions)
    *   Parámetros y valores de retorno
    *   Estructuras condicionales (`if`, `else if`, `else`)
    *   Operadores ternarios
    *   Anidamiento de condiciones
*   **Actividades específicas:**
    *   Crear 4 funciones tradicionales que implementen lógica condicional compleja.
    *   Desarrollar 3 arrow functions con diferentes estructuras de control.
    *   Implementar 3 programas que utilicen operadores ternarios para decisiones simples.

**IV. MÓDULOS Y ESTRUCTURA DE CÓDIGO** (10 Ejercicios)

*   **Temas a desarrollar:**
    *   Exportación de funciones y variables (`export default`, `export` nombrado)
    *   Importación de módulos (`import`)
    *   Organización de código en múltiples archivos
    *   Uso de alias en importaciones
    *   Estructuración de proyectos JavaScript
*   **Actividades específicas:**
    *   Crear 5 pares de archivos que demuestren exportación e importación por defecto.
    *   Desarrollar 3 conjuntos de módulos con exportaciones nombradas.
    *   Implementar 2 proyectos estructurados con múltiples archivos JavaScript.

**V. MANEJO DE ARCHIVOS CON FS** (10 Ejercicios)

*   **Temas a desarrollar:**
    *   Operaciones CRUD con archivos de texto
    *   Uso de `fs.writeFile` para creación de archivos
    *   Implementación de `fs.appendFile` para agregar contenido
    *   Lectura de archivos con `fs.readFile`
    *   Verificación y eliminación de archivos (`fs.existsSync`, `fs.unlinkSync`)
*   **Actividades específicas:**
    *   Desarrollar 3 programas para creación y escritura de archivos.
    *   Crear 3 ejercicios de lectura y procesamiento de contenido de archivos.
    *   Implementar 2 programas para modificación (update) de archivos existentes.
    *   Desarrollar 2 ejercicios de verificación y eliminación de archivos.

### **REQUISITOS DE ENTREGA**

*   **Formato de presentación:**
    *   Todos los ejercicios deben entregarse en archivos JavaScript individuales.
    *   Los archivos deben nombrarse siguiendo el patrón: `tema_ejercicioX.js`
    *   Incluir comentarios explicativos en cada ejercicio.
